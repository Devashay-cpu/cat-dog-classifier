# 🐱🐶 Cat vs Dog Classifier

## 📌 Project Overview

This project is a Convolutional Neural Network (CNN) based image classifier that predicts whether an image is of a **cat** or a **dog**.

## 🚀 Features

* Image classification using CNN
* Real-time prediction on custom images
* Clean prediction function
* Accuracy & loss visualization

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* Matplotlib

## 📊 Model Performance

* Accuracy: ~90%+
* Binary classification (Cat vs Dog)

## ▶️ How to Run

```bash
pip install -r requirements.txt
```

```python
predict_image('image.jpg', model)
```
